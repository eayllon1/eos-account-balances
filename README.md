## Modular 11-2019

# EOS Account Balances

Shows all current account balances for a given EOS account.

## 1. Getting Started


### 1.1 Installation

Clone, then install dependencies in the server and client.

```
cd server
npm install
```

```
cd client
npm install
```

### 1.1 Running

Run the server first, then the client.

Sever:

```
npm run dev
```

Web:

```
npm run web
```

Mobile:

```
npm run start
```
